## 🏁 Getting Started

### Prerequisites
- **Node.js** (v18 or higher)
- **Package Manager** (npm, yarn, or bun)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sajid-7756/sajid-protfolio.git
   cd sajid-protfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Development Mode:**
   ```bash
   npm run dev
   ```

4. **Production Build:**
   ```bash
   npm run build
   ```
